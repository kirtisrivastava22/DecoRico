# DecoRico 

**DIY Decoration Materials Shopping Application**

A mobile application for browsing and purchasing DIY decoration materials, tools, and supplies for home decoration projects.

## Getting Started

### Prerequisites
- Node.js installed
- MongoDB running locally or connection string ready

### Installation
```bash
# Clone the repository
git clone https://github.com/kirtisrivastava22/DecoRico.git

# Navigate to project directory
cd DecoRico

# Install dependencies
npm install

# Start the application
nodemon app.js
```

##  Features

- Browse DIY decoration materials by category
- Search and filter products
- Add items to cart and checkout
- User authentication and profiles
- Order tracking
- Product reviews and ratings

##  Technology Stack

- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Frontend**: HTML, CSS, JavaScript
- **Authentication**: JWT

##  Main Categories

- Wall Decor
- Lighting
- Furniture Accessories
- Craft Supplies
- Storage Solutions
- Tools & Hardware

##  Usage

1. Start the server with `nodemon app.js`
2. Open your browser and navigate to the local server
3. Register/Login to start shopping
4. Browse products and add to cart
5. Complete checkout process
